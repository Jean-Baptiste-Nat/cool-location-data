Installation

Supporte script tag, ES Modules, npm package, CDN et API REST Firebase.

npm install cool-location-data
<script src="https://cool-location-data.web.app/location-data.js"></script>

Utilisation

window.CoolLocationData ESM import Node.js compatible 

                    import { getCountryData, getRegion, getShippingZone } from "cool-location-data";
                    const country = getCountryData("CA");
                    const region = getRegion("CA", "CA-QC");
                    const shipping = getShippingZone("CA", "CA-QC");

Helpers JavaScript

Helpers implementes: getAllCountries, getCountryData, getCountryName, getCountryFlag, getCountryCurrency, getCountryLanguages, getCountryPhoneCode, getCountryTimezones, getCountryCapital, getRegions, getRegion, getRegionName, getRegionCode, getMetroAreas, getShippingZone, getTaxZone, getDistanceBetweenRegions, searchCountry, searchRegion, searchCity.

Shipping, taxes et zones

Chaque pays/region embarque shippingZone, taxZone/taxRegion, timezone, langue principale et metriques logistiques pour e-commerce, multi-store et marketplace.

Stripe, paiements et checkout

Le mapping country/region/currency/langues facilite la normalisation des adresses, le choix de devise, la fiscalite et la coherence du tunnel de paiement.
